Docs for Hekinav Oikotie Proxy API

This API requires authentication

This API uses the Hekinav API Token system. To access, create an application with this API in the API portal


Endpoints

Docs: /

Returns this page

Random property: /randomProperty?locations={locations}

Returns a random property in the specified region

Parameters:

locations: [number, number, string][] (example: [[39, 6, "Espoo"]])

General proxy: /proxy/{path}

Returns a random property in the specified region

Parameters:

path: any oikotie API path (docs) (example: /search?locations=[[39,1, "Espoo"]]&limit=3)

Headers: /headers

Returns headers that can be used to authenticate Oikotie API requests

Parameters:

none
